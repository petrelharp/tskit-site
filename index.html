The open source tree sequence community

Welcome to tskit.dev, the home of the open source tree sequence community! Tree sequences are a powerful way to represent genetic ancestry and DNA sequence data, leading to efficient storage and computation as well as new biological insights. This site is a collection of resources to help you learn about tree sequences and available software, and to help you join the online community of open source developers that makes it all possible.

This site is under construction (think of it as an early alpha release!)
